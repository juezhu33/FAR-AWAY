# Travel List - 旅行清单

A simple and intuitive travel packing list application built with React. This app allows users to add items they need for their trip, mark them as packed, and sort the list by different criteria.

一个使用 React 构建的简洁直观的旅行打包清单应用。该应用允许用户添加旅行所需物品，标记为已打包，并按照不同标准对列表进行排序。

## Features - 功能

- Add items with quantity - 添加带数量的物品
- Mark items as packed - 将物品标记为已打包
- Sort items by input order, description or packed status - 按输入顺序、描述或打包状态排序物品
- Clear entire list - 清空整个列表
- Responsive design - 响应式设计

## Technologies Used - 使用的技术

- React
- JavaScript (ES6+)
- CSS3
- HTML5
